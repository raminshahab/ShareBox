## Development Environment Requirments [ Valet ]

Homebrew php72 [ php 7.2.4 ]
```sh
brew install homebrew/php/php72
```
Install Valet with Composer 
```sh
$ composer global require laravel/valet
```
Run Valet install 
```sh
$ valet install 
```
